Features

Interactive Menus

Every command uses arrow-key navigation. Browse profiles, tools, models, history, and settings with a consistent picker UX.

43+ Built-in Tools

System info, file operations, git workflows, shell execution, web requests, Docker, SSH, clipboard, and more. Categorized and toggleable.

4 LLM Providers

Ollama (local, free), GitHub Models (free with Copilot), OpenAI (GPT-4o), and Anthropic (Claude). Switch instantly with /profile.

Developer Environment

/dev audits your setup across 12 categories: languages, AI tools, editors, cloud CLIs, containers, databases, and more. One-click installs.

Streaming + Markdown

Responses stream in real-time with glamour markdown rendering. Shell commands show output line-by-line as they run.

macOS Signed + Notarized

Native Apple Silicon DMG with drag-to-install. Self-updates via emprise update. Also available for Linux and Windows.

Commands

Chat

  • /clear reset conversation
  • /cls clear screen
  • /name name conversation
  • /save save conversation
  • /history browse past conversations
  • /copy copy last response to clipboard
  • /export export as markdown

LLM

  • /model switch model (picker or /model gpt-4o)
  • /profile switch LLM profile
  • /config settings menu
  • /tools manage tools by category
  • /stats usage heat map per model
  • /setup add a new LLM provider

Developer

  • /dev environment audit + install
  • /dev inventory full text report
  • /font Nerd Font setup
  • /ask quick one-off question
  • /run run shell command
  • /about version, safety, practices

CLI Flags

  • emprise "question" single-shot answer
  • emprise -c continue last conversation
  • emprise --resume name resume by name
  • emprise --list list conversations
  • emprise update self-update
  • emprise install reinstall
  • cat file | emprise "explain" pipe mode

Safety

Blocked Commands

rm -rf /, dd, mkfs and other destructive operations are hard-blocked.

Protected Paths

/etc, /usr, /System, and home dotfiles are protected from modification.

Inline Visibility

Every tool call is shown with its arguments before running. Nothing hidden.

Context Limits

File reads capped, output truncated, tool chains limited to 10 rounds.

No Exfiltration

Your code is only sent to the LLM provider you configured. Nothing else.

Read-Only Default

Most tools (system info, git status, file reading) are read-only and always safe.
